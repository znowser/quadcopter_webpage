Welcome to the quadcopter GUI

Live Video

Come and watch the quadcopter in action. See whatever the quadcopter sees. A camera module is mounted onto the quadcopter and can send a stream to our web server.

Dynamic data updates

The quadcopter can send a stream of dynamic data to our server, making it possible to see everything about the quadcopter dynamically. Everything from current battery level to engine load.

Coordinates

This GUI makes the user able to see where the quadcopter is located, using a map module and live coordinates from the user. The user will also be able to create routes for the quadcopter through this map.

Control it

Not only is the user able to watch a live stream from the quadcopter, the user will also be able to stear the quadcopter.

Quadcopter
