Mobile GUI

The quadcopter mobile GUI is a GUI more suitable for mobile devices.

Welcome to the quadcopter GUI

Live Video

Come and watch the quadcopter in action. See whatever the quadcopter sees. A camera module is mounted onto the quadcopter and can send a stream to our web server.

Dynamic data updates

The quadcopter can send a stream of dynamic data to our server, making it possible to see everything about the quadcopter dynamically. Everything from current battery level to engine load.

Coordinates

This GUI makes the user able to see where the quadcopter is located, using a map module and live coordinates from the user. The user will also be able to create routes for the quadcopter through this map.

Control it

Not only is the user able to watch a live stream from the quadcopter, the user will also be able to stear the quadcopter.

About

Quadcopter

Quadcopter is a project driven by five students on Linköpings Tekniska Högskola. It is about building a functional quadcopter.

Web GUI

Controls for Quadcopter data

Which data to show:
Battery

Battery shows all cells individually aswell as complete battery percentage.
Click on data in the graph to pause it.

Show battery power

Engine load

Showing the engine load on the different engines.
Click on data in the graph to pause it.

Show engine load

Temperature

Showing temperature values over an interval of 10 seconds.
Click on data in the graph to pause it.

Show temperatures

Altitude

Showing altitude values over an interval of 10 seconds.
Click on data in the graph to pause it.

Show altitude

3d model

Showing a dynamic 3d model over the quadcopter.
Click on data in the graph to pause it.
This is a feature hopefully coming in the future.
