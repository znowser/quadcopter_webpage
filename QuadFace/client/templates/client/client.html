Welcome!

This site is to be used as a Web GUI for a quadcopter some students from Linköpings Tekniska Högskola are building. The developments of this site is being driven by two of the students involved in former mentioned group. We are not only building this site for the quadcopter but also for a class named TDDD27 Advanced Webprogramming at Linköpings Tekniska Högskola. When this site is done, a user will be able to look at a live feed coming from the quadcopter. The user will be able to look at live data coming from the quadcopter. The user will also eventually be able to control the quadcopter from this GUI.
